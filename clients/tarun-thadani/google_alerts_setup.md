## How to set up each alert

1. Go to **google.com/alerts**
2. Sign in with your Google account
3. In the search box, type the query exactly as shown above (including quotes)
4. Click **Show options**
5. Set:
   - **How often**: As-it-happens (or Once a day for lower-volume ones)
   - **Sources**: Automatic
   - **Language**: Any
   - **Region**: Any
   - **How many**: All results
   - **Deliver to**: tarun@dharte.com
6. Click **Create Alert**
7. Repeat for each query in the table above
